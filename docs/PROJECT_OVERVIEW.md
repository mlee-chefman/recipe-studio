# CHEF iQ Studio - Project Overview

## Competition Context
This is a mobile app for the CHEF iQ Studio App Challenge, a 2-month competition to build a recipe authoring app that enables users to create recipes for the CHEF iQ platform.

## Project Goals
Build a standalone mobile app that enables creative home cooks to author recipes and publish them in the main CHEF iQ app, where others can cook them using Guided Cooking.

## Timeline
- **Start**: September 15, 2025
- **Midpoint Check-in**: October 20-24, 2025
- **Submission**: November 14, 2025
- **Presentations**: November 17-21, 2025

## Team
- Solo developers working off-hours
- Estimated 20-30 hours/week
- Total: ~160-240 hours of development time

## Target Users
- Creative home cooks who love sharing knowledge
- CHEF iQ power users who know the appliances well
- Community builders who enjoy helping others

## Key Success Metrics
1. **Functional app** that creates, edits, and publishes recipes
2. **Clean, intuitive UX** for the core recipe creation flow
3. **Demo-ready** with 3-5 example recipes showcasing different appliances
4. **Clear integration path** to CHEF iQ ecosystem

## Deliverables Required
1. **Deployed App** - Available on TestFlight (iOS) and/or App Tester (Android)
2. **Code Repository** - Hosted on GitHub with clear setup instructions
3. **Written Documentation** - PDF detailing design process and business considerations
4. **Presentation** - Live demo and Q&A during judging week

## Judging Criteria (75 points total)
- **Idea (20 pts)**: Creativity, Feasibility, User Impact, Business Impact
- **Execution (40 pts)**: User-Centered Design, Design Polish, Software Quality, App Completeness
- **Deliverables (10 pts)**: Documentation, Presentation, Deployment
- **Midpoint Check-in (5 pts)**: Progress demonstration

## Awards
- 1st Place: $20,000
- 2nd Place: $10,000
- 3rd Place: $5,000
- Category Awards: $2,500 each (Most Innovative, Best Use of AI, Best UX, Best Business Case)

## Key Constraints
- Must integrate with CHEF iQ appliances (MiniOven, SmartThermometer, PressureCooker)
- Recipes must support Guided Cooking features
- Development primarily outside working hours
- 2-month timeline for full app development
